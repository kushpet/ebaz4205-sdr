План работ по написанию программного обеспечения для самодельного SDR трансивера на базе платы контроллера майнера EBAZ4205. 
Есть плата EBAZ4205, её принципиальная схема в файле ebaz4205_schematic_color.pdf. Есть плата с ADC, её схема в файле AD9226-V1.2-board.pdf. Есть плата с DAC, схема DAC904-board.pdf. Соединения планы EBAZ4205 через разъемы DATA1, DATA2 с платой DAC и через разъем DATA3 с платой ADC описаны в файле EBAZ4205-ADC-DAC.md. В колонке "Разъем" название сигнала на EBAZ4205, в колонке "Плата" название сигнала на соответствующей плате. 
Цель - написать программное обеспечение, которое позволит использовать данное оборудование как SDR (software defined radio) приёмник и передатчик. Под управлением ПО SDRAngel. Использовать как FPGA, так и процессорную систему Zynq7010. Можно с использованием Embedded Linux, но лучше Bare Metall. Связь с SDRAngel через Ethernet разъем на EBAZ4205. Частота дискретизации ADC/DAC 60 МГц.
На первом этапе определи, достаточно ли тебе информации. Определи документы, которые необходимо дополнительно подгрузить в это пространство, поскольку они будут использоваться при последующих запросах.
